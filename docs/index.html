Double-click Setup · once

Sets up everything the app needs. A window opens, runs for about a minute, and says when it's done. One time only.

First launch: macOS will say it “can't verify the developer.” Instead of double-clicking, right-click Setup → Open → Open. Once per file, then double-click works.
Wait for “Setup complete,” then close the window.

Your everyday step. A window opens and the downloader appears in your browser. Paste links, download.

Keep the window open while you work; close it to stop the app.
Opens as text instead of running? See “It opened as text” below.

3✓

Install · one time

Type cd and a space, then drag the folder onto the Terminal window and press return.

Then paste this block and press return:

Terminal
python3 -m venv venv
source venv/bin/activate
python3 -m pip install flask yt-dlp -i https://pypi.org/simple/
mkdir -p bin
curl -sL -o ffmpeg.zip https://evermeet.cx/ffmpeg/getrelease/zip
unzip -q ffmpeg.zip -d bin/
rm ffmpeg.zip
chmod +x bin/ffmpeg
No red ERROR lines means it worked.

If something looks wrong

“Can't verify the developer” +
Right-click the file → Open → Open, once per file. If your Mac is IT-locked and offers no “Open” option, that's a Santa/MDM block — ask IT to allowlist the app, or use the Manual tab.
It opened as text instead of running +
Open Terminal, type chmod +x (trailing space), drag the file in, press return. Double-click works after. Running Setup this way also re-arms Start Downloader.
“pip not found” +
Use python3 -m pip instead of pip.
“Address already in use” +
It's already running — open 127.0.0.1:5001.
Download fails with “403” +
Handled in this version. If it persists, flag it to whoever set this up.
