YouTube Downloader · Mac setup

Universal Downloader guide

Download the app (.zip)

Tap a step to open it.

1 Unzip it

Unzip the file you downloaded.

YouTube Downloader.command is the file that opens the app every time, so put it somewhere you'll remember.

2 Double-click it

Double-click YouTube Downloader.command. The first time, macOS blocks unsigned tools, so it won't open yet — that's expected.

Click Done.

3 Allow it

  1. Open Spotlight Search by clicking the magnifying glass in the top-right.

  2. Type Privacy into the search bar and click the Privacy & Security suggestion. This opens Privacy & Security settings.

  3. In Privacy & Security, scroll all the way down to Security. On the prompt “YouTube Downloader.command was blocked to protect your Mac,” click Open Anyway. You'll be prompted once more — click Open Anyway again. If you're asked for your password, enter it.

4 Allow any pop-ups

As it starts up the first time, you may see one or two more prompts — just allow each:

  1. “Terminal would like to access files in your Downloads folder” → click Allow.

  2. “The python3 command requires the command line developer tools…” → click Install, then Agree to the license. It keeps going on its own once that finishes — no need to reopen anything.

Don't see any pop-ups? They can hide behind your browser or other open windows, so check there first. You may also already have Python installed, which is fine. If nothing appears, just skip ahead to the next step.
5 You're set

It downloads and sets up (about a minute the first time), then your browser opens with the downloader. Paste links and go. Keep the Terminal window open while you work; close it to stop. After this first time, you never touch Settings again.
